## What does this code do?

This code serves to generate a Provably Fair dice roll.

## What is Provably Fair?

"[Provably Fairness](https://en.wikipedia.org/wiki/Provable_fairness#:~:text=Provably%20fairness%20is%20a%20technology,do%20not%20cheat%20their%20customers) is an algorithmic approach that can be scrutinized and tested to ensure fairness. Typically employed in online cryptocurrency-based gambling, Provably Fair technology aims to guarantee that online casinos operate without deceiving their customers." - Quoted from [Provably Fairness](https://en.wikipedia.org/wiki/Provable_fairness#:~:text=Provably%20fairness%20is%20a%20technology,do%20not%20cheat%20their%20customers)

## How to verify if the rolls are correct?

Several websites facilitate the verification of Provably Fair rolls. For example, you can visit the [Provable Fair Faucet Game](https://provablefair.altervista.org/faucet-game/primedice/?doing_wp_cron=1705957378.1398849487304687500000).
